Z140 MDIS Driver Documentation

This is the documentation of the MDIS5 low-level driver for the MEN 16Z140 Frequency Counter IP core.


Functional Description


IP Core

The 16Z140 is a Frequency Counter IP core that performs different analysis on either a single signal input or on a quadrature encoded input with two signals.

The Frequency Counter IP core supports the following functions:

  • Frequency Measurement of Signal A
  • Frequency Measurement of Signal B
  • Direction Detection
  • Standstill Detection
  • Rolling Detection
  • Distance Measurement
  • Built-In Test Frequency Generator


Driver

The driver supports controlling of the Frequency Counter via M_setstat() and M_getstat(), see Getstat/Setstat codes.

When the first path is opened to an 16Z140 device, the IP core and the driver are being initialized.
